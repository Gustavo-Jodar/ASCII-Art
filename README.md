# ASCII Art Generator


## Description
The project wants to generate ASCII Art from different inputs with C++ by using [OpenCV Library](https://opencv.org/).

![alt text](images/readme.png)

### Features
Accepted input:
- File Image in PNG;
- File GIF in GIF;
- File Video in MP4;


## Installation
Follow the [install OpenCV](https://docs.opencv.org/4.x/d7/d9f/tutorial_linux_install.html) documentation.

## Roadmap
This project had incremental goals:
1. Open a file in:
   - PNG;
   - GIF;
   - MP4;
    
    And converting into an ASCII Art;

2. Use real-time input from the computer's webcam;

3. Save into a `.txt` with:
   - Custom size;
   - *WhatsApp*; 
   - *Telegram*;
